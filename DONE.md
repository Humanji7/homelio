## Acceptance Criteria For The Full Test Assignment

- Есть CLI или мини-сервис, который по тестовым JSON генерирует 3 заполненных DOCX.
- Для каждого документа описан машиночитаемый маппинг с указанием источников полей.
- В `output` лежат 3 результата генерации на тестовых данных.
- В `README.md` есть пошаговый локальный запуск и проверка.
- В `notes` есть короткая записка по архитектуре, роли детерминированной логики и путям масштабирования.
- Проект опубликован в публичном GitHub-репозитории.
- Подготовлен чистый артефакт сдачи без `.venv`, кешей и служебного мусора.

## Validation Required Before Completion

- Запуск команды генерации проходит локально на чистом окружении.
- Тесты на ключевые функции маппинга и заполнения проходят.
- Результат хотя бы по основным полям сопоставлен с эталонными примерами.
- Отсутствующие поля обрабатываются без падения и без выдуманных значений.
- Публичный remote доступен, а рабочая ветка запушена.
- ZIP-архив открывается и содержит ожидаемую структуру проекта.
