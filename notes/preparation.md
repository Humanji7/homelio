# Preparation Notes

## What We Know Already

- Все 3 шаблона содержат явные текстовые плейсхолдеры в квадратных скобках.
- Для первого документа плейсхолдеры напрямую соответствуют данным из `company_profile_example.json`.
- Во втором и третьем документах есть смесь реквизитов компании, данных тендера, расчетных значений и системных полей вроде даты.
- Краткое описание логики из пакета подтверждает правильный вектор: детерминированный маппинг важнее LLM, AI допустим только как опциональный бонус для извлечения данных.

## Chosen Base Stack

- `Python + uv` для локального, быстро поднимаемого CLI-решения.
- `typer` для воспроизводимой точки входа.
- `lxml` + прямой доступ к OpenXML внутри DOCX для детерминированного заполнения без ручной постобработки.
- `PyYAML` как удобный формат для машиночитаемых маппингов.
- `pytest` и `ruff` для минимальной инженерной дисциплины с самого начала.
